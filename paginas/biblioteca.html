Biblioteca UFPR - Palotina

Projeto desenvolvido para a biblioteca da Universidade Federal do Paraná (UFPR) - Setor Palotina. Atuei como bolsista na biblioteca entre os anos de 2017 e 2019. O intuito do software é fazer a estatística diária de frequência e consulta da biblioteca.

Tecnologias utilizadas

  • HTML
  • CSS - Materialize
  • JavaScript - JQuery
  • PHP
  • SQL - MariaDB
