Tela inicial

Essa é a nossa interface inicial do programa "Consulta Local"

Com a tela mapeada acredito que fica mais fácil, irei explicar item por item.

Item 1. O item destacado mostra onde o programa irá receber o código para fazer a gravação no diretório padrão.

Item 2. O item destacado mostra a localização do botão que deverá ser pressionado quando o código é devidamente inserido, sua tecla de atalho é ENTER.

Item 3. O item destacado mostra onde você poderá acessar os arquivos gravados, ao clicar em cima dessa opção com o mouse, uma nova janela será aberta. Como na imagem abaixo:

Nesta imagem vemos a opção PASTA que sua função é de abrir a pasta onde está sendo salvo os códigos de barras, e a opção ARQUIVO que quando clicada abre o arquivo que está sendo atualizado.

Item 4. Voltando para o mapeamento da tela inicial, este item tem a função de exibir o código registrado anteriormente e caso de erro (se o código inserido for menor que 8 digitios), no mesmo local exibirá uma mensagem que o código é INVÁLIDO.

Item 5. Chegamos no rodapé do programa, nele está contido as informações sobre a versão do software e a data retirada do relógio do computador.

Opções

A pequena janela onde mostra CONTADOR e HISTÓRICO pode ser acessada pressionando com o mouse o botão USUÁRIOS

O contador de usuários, pode ser acessado pela aba USUÁRIOS na tela inicial, ou pelo seu atalho F1.

A imagem apresentada acima mostra o mapeamento da janela USUÁRIOS separada por itens.

Item 1. Mostra o contador do periodo atual, na imagem está no perido da manhã, mas por exemplo, se tivesse de tarde o contador seria resetado e começado um novo.

Item 2. Adiciona um número no contador.

Item 3. Remove um número do contador (para caso de registrar por engano.

Item 4. Possibilita definir um número para ser adiconado, por exemplo: Entrou 5 pessoas na biblioteca, eu posso alterar o 0 para 5 e pressionar o botão + que ele irá adicionar às 5 pessoas automaticamente, mesma coisa equivale para o botão - que ao invés de adicionar, ele subtraí.

Item 5. Este botão permite sincronizar os arquivos.

Item 6. Mostra a quantidade de pessoas registradas nos devidos períodos.

O sistema de busca, pode ser acessado pela aba USUÁRIOS na tela inicial, ou pelo seu atalho F2.

A imagem apresentada acima mostra o mapeamento da janela SISTEMA DE BUSCA separada por itens.

Item 1. Permite você digitar um dia em específico (DD-MM-AAAA), exemplo: 24-11-2017.

Item 2. Busca o arquivo que tem o endereço da data digitada no Item 1.

Item 3. Salva as alterações do arquivo no Item 4.

Item 4. Mostra detalhadamente a estatística do dia e é possivel alterar os dados.

Item 5. Recebe um mês para ser buscado pelo botão do Item 7.

Item 6. Recebe o ano para ser buscado pelo botão do Item 7.

Item 7. Botão que recebe o Mês e o ano digitados nos outros itens para atualizar a tabela do Item 8.

Item 8. Mostra um histórico detalhado com total de cada dia e um total geral do Mês e Ano digitados no Item 5 e Item 6.

O que causar problema?
  • Falta de internet
  • Má sincronização com o Google Drive
  • Desacordo entre os operantes e registrarem uma pessoa duas vezes
  • Horário do computador
Como evitar estes problemas?
  • Manter a hora do computador atualizada
  • Verificar se a sincronização com o drive está OK
  • Acordo entre os operadores
